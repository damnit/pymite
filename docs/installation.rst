Installation
============

.. currentmodule:: pymite

Install pymite with ``pip`` ::

    pip install pymite 


The development version can be downloaded from `its page at GitHub
<https://github.com/damnit/pymite>`_. ::

    git clone https://github.com/damnit/pymite.git
    cd pymite
    python setup.py develop


pymite has no dependencies apart of the stdlib.
Compatible Python versions: 3.3, 3.4 or 3.5.
